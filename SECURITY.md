# Security Policy

## Reporting a vulnerability

Email `hello@nothumansearch.ai` with subject line prefix **`SECURITY:`** and a short description of the issue, impact, and any reproduction steps.

- We commit to responding within **48 hours**.
- Please do not open public GitHub issues for security reports.
- We do not currently run a bug bounty program — we cannot pay for reports, but we will credit reporters who want attribution once a fix ships.

## Scope

Everything in this repository, including example code and workflows. These are reference examples; if you find a pattern here that would be unsafe in production, we want to know.
